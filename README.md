# RoboLab Inventory App

This is a single-page Robotics Lab Inventory application.

## How to Run

You can run this application using Node.js, Python (Standard), Python (Flask), or Streamlit.

### Option 1: Streamlit (Recommended if you prefer Python Data Apps)
If you have Streamlit installed (`pip install streamlit`):

```bash
streamlit run streamlit_app.py
